# YESH ENTERPRISES – Business Website

A fully responsive, modern business website for **YESH ENTERPRISES**, a professional toner refill and printer service company based in Mumbai.

---

## 📁 Project Structure

```
├── index.html      # Main HTML file
├── style.css       # All styles (responsive, animations)
├── script.js       # JavaScript (nav, typing, FAQ, forms)
└── README.md       # This file
```

---

## 🚀 Features

### Design
- CMYK-inspired color palette (Cyan, Magenta, Yellow, Black)
- Floating pill-shaped navbar with glassmorphism effect
- Hero section with animated ink blob backgrounds
- Smooth scroll-reveal animations on all sections

### Navigation
- Floating rounded sticky navbar
- Desktop: hover dropdown for Services with animated chevron
- Mobile: slide-down menu panel with tap-to-expand Services accordion
- Animated hamburger → X button
- Backdrop overlay with body scroll lock
- Escape key and overlay-click to close

### Sections
| Section | Description |
|---|---|
| Hero | Typing effect tagline, CMYK pills, floating printer SVG |
| Services | 4 cards — Toner Refills, Printer Repair, Maintenance, Cartridge Supply |
| About | Proprietor profile, stats badge |
| Gallery | 6 placeholder cards with hover zoom |
| Testimonials | 4 customer review cards with star ratings |
| FAQ | Collapsible accordion with smooth animation |
| Contact | Info cards + validated contact form |
| Footer | Quick links, social icons, newsletter subscription |

### JavaScript Features
- Typing effect cycling through 5 service phrases
- Scroll-triggered fade-in with staggered delays
- FAQ accordion (one open at a time)
- Contact form validation (name, email format, message)
- Newsletter email validation
- Back to Top button
- Active nav link highlight on scroll

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 1024px` | Full desktop layout |
| `≤ 1024px` | Stacked about section, 2-col footer |
| `≤ 900px` | Single column hero, stacked contact |
| `≤ 640px` | Mobile nav panel, single column everything |

## 🌐 Usage

No build tools required. Just open `index.html` in any browser.

```bash
# Clone or download, then open directly
open index.html
```

> Requires an internet connection to load Font Awesome icons from CDN.
